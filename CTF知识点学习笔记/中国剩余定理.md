# 中国剩余定理（CRT）学习笔记

## 一、什么是中国剩余定理

中国剩余定理（Chinese Remainder Theorem, CRT）是数论中一个基本定理，用于求解**一元线性同余方程组**：

```
x ≡ a₁ (mod m₁)
x ≡ a₂ (mod m₂)
...
x ≡ aₖ (mod mₖ)
```

其中所有模数 **m₁, m₂, ..., mₖ 两两互质**（即 gcd(mᵢ, mⱼ) = 1）。

该定理保证在模 `M = m₁·m₂·…·mₖ` 下有**唯一解**。

**白话解释**：给你一系列“除以某个数余几”的条件，求一个满足所有条件的最小非负整数。

---

## 二、数学公式（求解方法）

设 `M = ∏mᵢ`，`Mᵢ = M / mᵢ`，`tᵢ = Mᵢ⁻¹ mod mᵢ`（即 Mᵢ 在模 mᵢ 下的逆元）。

则方程组的解为：

```
x = (Σ aᵢ · Mᵢ · tᵢ) mod M
```

**验证**：对任意 j，当 i ≠ j 时，Mᵢ 含因子 mⱼ，所以该项模 mⱼ 为 0；当 i = j 时，Mⱼ·tⱼ ≡ 1 (mod mⱼ)，所以 x ≡ aⱼ (mod mⱼ)。

---

## 三、使用条件

1. **模数互质**：这是 CRT 成立的前提。如果不互质，需要用**扩展的中国剩余定理**（可处理非互质情况），但 CTF 中多数情况模数互质（或通过选择样本保证互质）。
2. **同余方程组必须一致**：即方程组有解（通常题目保证有解）。
3. **需要至少一个未知数 x**，且所有同余式都是线性形式。

---

## 四、何时使用 CRT？

### 1. 求解同余方程组
这是 CRT 的本职工作。例如：
- 已知一个数除以 3 余 2，除以 5 余 3，除以 7 余 2，求这个数。
- 在 RSA 广播攻击中，我们有 `cᵢ ≡ m^e (mod nᵢ)`，要求出 `m^e` 的精确值（不取模），这时将 `m^e` 视为未知数，用 CRT 合并同余式。

### 2. 加速模幂运算（分解指数）
如果模数 `n = p·q`，可分别计算 `a mod p` 和 `a mod q`，然后用 CRT 合并结果得到 `a mod n`。这在某些密码学计算中用于加速。

### 3. 秘密共享（Shamir 秘密共享等）和插值
CRT 可用于多项式插值（类似于拉格朗日插值），但更常见的是同余式组合。

### 4. 组合数取模（大组合数取多个小模数）
当模数不是质数时，可分解为质数幂的乘积，分别计算组合数模这些质数幂，再用 CRT 合并。

## 五、CTF 中的典型应用

### 1. RSA 广播攻击（Håstad's Broadcast Attack）
- **场景**：同一明文 `m`，用小加密指数 `e`（如 3）加密给多个不同的模数 `nᵢ`，得到多个密文 `cᵢ`。
- **方程组**：`cᵢ ≡ m^e (mod nᵢ)`。
- **目标**：恢复 `m`。
- **解法**：用 CRT 解出 `X ≡ m^e (mod N)`，其中 `N = ∏nᵢ`。由于 `m^e < N`，`X` 就是 `m^e` 的精确值，再开 `e` 次方得 `m`。

### 2. 共模攻击（Common Modulus Attack）
- **场景**：相同的 `n`，不同的 `e₁` 和 `e₂` 加密同一明文 `m`，得到 `c₁`、`c₂`。
- **解法**：利用扩展欧几里得求 `e₁·u + e₂·v = 1`，则 `m = c₁^u · c₂^v mod n`。这里没有直接用 CRT，但思想类似（合并指数）。

### 3. 多素数 RSA（Multi-prime RSA）的解密加速
- 如果 `n` 是多个素数的乘积（如 `n = p·q·r`），解密时可用 CRT 分别模每个素数，再合并。

### 4. 根据多个同余条件恢复隐藏数据
- 例如，`flag` 被切分成多个块，每个块以不同的模数给出条件，最终用 CRT 拼回。

## 六、代码实现（Python 模板）

### 标准 CRT（模数互质）

```python
def crt(remainders, moduli):
    """
    参数：
        remainders : 余数列表 [a1, a2, ..., ak]
        moduli     : 模数列表 [m1, m2, ..., mk]（两两互质）
    返回：
        x : 满足所有同余式的最小非负整数
    """
    # 计算总模数 M
    M = 1
    for m in moduli:
        M *= m

    # 计算 x
    x = 0
    for a, m in zip(remainders, moduli):
        Mi = M // m
        inv = pow(Mi, -1, m)          # Python 3.8+ 支持模逆
        x += a * Mi * inv
    return x % M
```

### 扩展 CRT（模数不互质）

```python
def exgcd(a, b):
    if b == 0:
        return a, 1, 0
    g, x1, y1 = exgcd(b, a % b)
    return g, y1, x1 - (a // b) * y1

def crt_non_coprime(remainders, moduli):
    """
    合并两个同余方程：x ≡ a1 (mod m1)，x ≡ a2 (mod m2)
    返回合并后的 (a, m)
    """
    a1, m1 = remainders[0], moduli[0]
    for a2, m2 in zip(remainders[1:], moduli[1:]):
        g, p, q = exgcd(m1, m2)
        if (a2 - a1) % g != 0:
            raise ValueError("No solution")
        lcm = m1 // g * m2
        t = ((a2 - a1) // g * p) % (m2 // g)
        a1 = (a1 + m1 * t) % lcm
        m1 = lcm
    return a1 % m1, m1
```

## 七、常见题型模板

### 题型1：广播攻击（标准 CRT）

**数据**：`e = 3`，`n_list`，`c_list`（长度为 3 或以上）。

**步骤**：
1. 检查 `len(c_list) >= e`（通常取 e 个即可）。
2. 调用 `M = crt(c_list, n_list)`。
3. `m = gmpy2.iroot(M, e)[0]`。
4. 将 `m` 转字符串。

**完整代码**（配合 gmpy2）：

```python
import gmpy2
from Crypto.Util.number import long_to_bytes

def hastad_broadcast(c_list, n_list, e):
    M = crt(c_list, n_list)
    m, exact = gmpy2.iroot(M, e)
    if exact:
        return long_to_bytes(int(m)).decode()
    return None
```

### 题型2：已知部分消息（Coppersmith + CRT）

有时明文 `m` 大部分未知，但已知高位（如 `flag{...}` 的格式），可构造多项式 `f(x) = (m_high + x)^e - c`，然后用 Coppersmith 求小根，可能涉及 CRT 模多个数。

### 题型3：解密加速（多素数）

设 `n = p·q·r`，`d` 已知，计算 `m_p = c^d mod p`，`m_q = c^d mod q`，`m_r = c^d mod r`，然后用 CRT 合并得到 `m mod n`。

## 八、注意事项与常见陷阱

1. **模数必须互质**：在广播攻击中，不同 `nᵢ` 通常互质（若存在公因数，则可直接分解）。若不互质，需用扩展 CRT。
2. **足够的样本数**：广播攻击要求密文数量 ≥ e，否则 `m^e` 可能大于 `∏nᵢ`，导致 CRT 恢复的值含有模数干扰，开方失败。
3. **逆元计算**：Python `pow(a, -1, mod)` 要求 Python 3.8+，低版本可用 `gmpy2.invert`。
4. **大整数类型**：`gmpy2.iroot` 返回的根是 `mpz`，需转为 `int` 再给 `long_to_bytes`。
5. **填充问题**：如果明文有 PKCS#1 等填充，解密后需要去除填充，否则会出现不可打印字符。在 CTF 中，通常明文就是 flag 字符串，无填充（或直接转字节）。

---

## 九、总结

中国剩余定理是数论和密码学中的核心工具，在 CTF 中主要用于：
- **广播攻击**：合并同余式，恢复精确数值。
- **加速计算**：将大模数分解为小模数分别计算。
- **秘密共享**：结合多个部分信息。

掌握 CRT 的数学原理和 Python 实现，就能快速应对相关题目。遇到“多组同余条件”时，第一时间想到 CRT，再结合题目特征判断是否可直接攻击。

**记忆口诀**：  
“模互质，乘积 M，每项除，逆元乘，求和模，即是解。”
